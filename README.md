# PRJ_SEG
シューティングゲーム　Project Segreta(セグレタ)

開発中でしたが色々残したまま一区切りとします。PRJ_RAINを下敷きにしています。

※MSX2++版は2025/1/26以前の版のz88dkでコンパイルして下さい。BGM用のファイルが正常に読めません。

※TOWNS版とMSX2++(V9968)版は今のところラスタースクロールはしません。

※MSX2++(V9968)版は改造版openMSX+FS-A1STイメージでのみ確認してます。拡張カートリッジの時はI/Oポートを修正する必要があります。

※TOWNS版はDJGPP12.2.0併用のmakeファイルになっています。旧版でコンパイルする時はsedの項を削ってください。

※title.sc5は縦を256ドットに拡張してあります。

※サウンドドライバはFM2リポジトリも参照してください。
