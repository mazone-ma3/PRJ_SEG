# PRJ_SEG
シューティングゲーム　Project Segreta(セグレタ)

![pyxel-20260319-080219](https://github.com/user-attachments/assets/fe0b5749-2448-4808-88f3-81553c2dcb0f)

PRJ_RAINを下敷きにしています。実機での動作は未検証未保証です。

※V9968(MSX2++)版は2025/1/26以前の版のz88dkでコンパイルして下さい。BGM用のファイルが正常に読めません。

※V9968版のラスタースクロールは若干怪しいです。

※TOWNS版のラスタースクロールはソフトウェア実装のため多少遅いです。

※V9968版は改造版openMSX+FS-A1STイメージでのみ確認してます。拡張カートリッジの時は専用バッチファイルを付けています。(暫定対応です)

※TOWNS版はDJGPP12.2.0併用のmakeファイルになっています。旧版でコンパイルする時はsedの項を削る等してください。

※title.sc5は縦を256ドットに拡張してあります。

※サウンドドライバはFM2リポジトリも参照してください。X68K版は途中で止まることがあります。

※完全移植ではありません。各版ごとに挙動が違います。
